## 🗓️ Lịch ôn tập gợi ý

### Tuần 1-2: Nền tảng
- Array, String, Hash Map, Two Pointers
- ~5 bài Easy/ngày

### Tuần 3-4: Core
- Stack, Queue, Linked List, Binary Search
- Trees (BFS, DFS, BST)
- ~3-4 bài Medium/ngày

### Tuần 5-6: Nâng cao
- Graph (BFS, DFS, Dijkstra, Topo Sort)
- Dynamic Programming
- ~2-3 bài Medium/ngày

### Tuần 7-8: Luyện thi
- Backtracking, Greedy, Heap
- Mock interviews
- ~2 bài Medium + 1 Hard/ngày
