# 📚 Cấu Trúc Dữ Liệu & Giải Thuật

> Tổng hợp kiến thức về Data Structures & Algorithms, có code minh họa và giải thích chi tiết.

## 📂 Cấu trúc thư mục

```
data-structures-algorithms/
├── 01-complexity-analysis/     # Big O, Time & Space Complexity
├── 02-array-string/            # Mảng, Chuỗi, Two Pointers, Sliding Window
├── 03-linked-list/             # Singly, Doubly, Circular Linked List
├── 04-stack-queue/             # Stack, Queue, Deque, Monotonic Stack/Queue
├── 05-hash-table/              # Hash Map, Hash Set, Collision Handling
├── 06-tree/                    # Binary Tree, BST, AVL, Red-Black Tree
├── 07-heap/                    # Min Heap, Max Heap, Priority Queue
├── 08-graph/                   # BFS, DFS, Dijkstra, Bellman-Ford, Topological Sort
├── 09-sorting/                 # Bubble, Selection, Insertion, Merge, Quick, Heap, Radix
├── 10-searching/               # Binary Search, Interpolation Search
├── 11-recursion-backtracking/  # Recursion, Backtracking, Permutations, Combinations
├── 12-dynamic-programming/     # Memoization, Tabulation, Classic DP Problems
├── 13-greedy/                  # Greedy Algorithm, Activity Selection, Huffman
├── 14-divide-and-conquer/      # Merge Sort, Quick Sort, Closest Pair
├── 15-trie/                    # Trie (Prefix Tree), Autocomplete
├── 16-union-find/              # Disjoint Set, Union by Rank, Path Compression
├── 17-segment-tree/            # Segment Tree, Fenwick Tree (BIT)
└── 18-advanced-patterns/       # Sliding Window, Two Pointers, Fast & Slow, Intervals
```

## 🎯 Cách sử dụng

Mỗi folder chứa:
- `README.md` — Lý thuyết, hình minh họa, độ phức tạp
- `examples/` — Code minh họa (JavaScript/TypeScript)
- `problems/` — Bài tập thực hành kèm lời giải
